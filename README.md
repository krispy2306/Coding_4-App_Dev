# Scrimble #


This folder contains all materials for submission for the App Development module.

> A community-focused mobile app for ameteur or professional esports players.
> Find scrimmages, discover tournaments, connect with other players and track competitive progress.

Built with React Native and Expo.



## Requirements:

- Node.js installed
- Expo Go installed on phone, or Android Studio emulator

- clone the repository
- run **npm install**
- run **npx expo start** 
- scan QR code on your phone, or press 'a' to run in Android emulator

- if project will not run in Android emulator, run with 'Cold Boot' via Android Studio.
